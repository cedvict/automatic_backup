New Features:

  • Fix ( .zip , .dump_) add and remove problems.
  • Improve Performance and reliability.

Configurations

  • Go to Settings / Technical / Back-ups / Configure Backup
  • Set your server configuration.
  • If you want to write backups to a remote location you should fill in the SFTP details.
  • Go to Settings / Technical / Automation / Scheduled actions
  • Search the action 'Backup scheduler'.
  • Set it active and choose how often you wish to take backups.
